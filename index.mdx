      <p className="text-base dark:text-gray-400 text-gray-600">
        Get Emdash running in minutes. Download, install your first agent, and start coding.
      </p>
    </div>

    <Steps>
      <Step title="Download Emdash">
        Download the latest release for your platform:
        
        <Tabs>
          <Tab title="macOS">
            ```bash
            # Apple Silicon
            curl -LO https://github.com/generalaction/emdash/releases/latest/download/emdash-arm64.dmg
            
            # Or via Homebrew
            brew install --cask emdash
            ```
          </Tab>
          <Tab title="Windows">
            ```bash
            # Download installer
            curl -LO https://github.com/generalaction/emdash/releases/latest/download/emdash-x64.msi
            ```
          </Tab>
          <Tab title="Linux">
            ```bash
            # AppImage
            curl -LO https://github.com/generalaction/emdash/releases/latest/download/emdash-x86_64.AppImage
            chmod +x emdash-x86_64.AppImage
            
            # Or Debian package
            curl -LO https://github.com/generalaction/emdash/releases/latest/download/emdash-amd64.deb
            sudo dpkg -i emdash-amd64.deb
            ```
          </Tab>
        </Tabs>
      </Step>

      <Step title="Install a coding agent">
        Install at least one CLI coding agent. Emdash supports 22+ providers:

        <CodeGroup>
        ```bash Claude Code
        curl -fsSL https://claude.ai/install.sh | bash
        ```
        ```bash Codex
        npm install -g @openai/codex
        ```
        ```bash Qwen Code
        npm install -g @qwen-code/qwen-code
        ```
        ```bash Amp
        npm install -g @sourcegraph/amp@latest
        ```
        </CodeGroup>

        <Info>See the full list of [supported providers](/features/providers) and their installation commands.</Info>
      </Step>

      <Step title="Open a project">
        Launch Emdash and add a git repository:
        
        1. Click **Add Project** in the sidebar
        2. Select a local git repository or clone a new one
        3. Emdash will detect installed agents automatically
        
        <Note>Emdash works with any git repository. Each task runs in an isolated worktree to prevent conflicts.</Note>
      </Step>

      <Step title="Create your first task">
        Start a new task with a coding agent:
        
        1. Click **New Task** 
        2. Select your preferred agent (Claude, Codex, etc.)
        3. Enter your prompt or attach a Linear/Jira ticket
        4. Watch the agent work in real-time with full terminal access
        
        <Tip>Create multiple tasks to run agents in parallel — each gets its own isolated worktree.</Tip>
      </Step>
    </Steps>
